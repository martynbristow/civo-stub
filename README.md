# CIVO API STUB

**Stub for [CIVO RESTful API](https://www.civo.com/api/kubernetes) (http://api.civo.com)**

Used for testing API Clients

## Setup

Requirements: `pip install -r requirements.txt`

## Deploy

The stub is a simple flask app.

Deploy: `python app.py`
